Tools I've Built

Live:

  • cmdk, an "access anything" shortcut for the terminal just like Notion, Slack, Discord, etc.
  • Wealthdraft, a financial planning tool for tracking your assets, monitoring net worth changes over time, estimating yearly tax, calculating changes needed to rebalance your portfolio, and more
  • FIRE Calculator, a Python script to calculate how much money you need today to never need to work again (FIRE)
  • FitPages, a web app for displaying data from the popular Fitnotes app
  • Overpowered Writing Tools, a collection of tools for managing a writing repo as defined in An Overpowered Writing System (TODO LINK)
  • CLI Journal, a CLI for writing and searching a repository of tagged journal entries
  • CLI Journal, an automated pipeline for pumping Brazilian bank account transactions into YNAB
  • Teact, a React-like framework for the Bubbletea TUI library
  • ActivityWatch Screentime Loader, a tool for regularly pushing ActivityWatch screentime data to a webhook
  • My Bash and Vim dotfiles
  • safebrew, a tiny Homebrew backup system
  • A script to download your Kindle highlights as JSON
  • A tool for querying the US Treasury website to get the value of a multiple savings bonds at a time

Dead:

  • Wunderpy2, Python bindings for Wunderlist (RIP)
  • Smooth Mechanism, my very first attempt at automating my GTD system
  • Unbox, a tool for symlinking things on your filesystem (e.g. your dotfiles) into Dropbox so they're always up-to-date
  • file-vacuum, which attempted to do what sftp does before I knew sftp existed
